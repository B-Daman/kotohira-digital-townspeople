## 1. サイト基本情報

### 1.1 サービス概要

| 項目 | 内容 |
|------|------|
| **正式名称** | 琴平デジタル町民（KOTOHIRA Digital Town Residents） |
| **運営主体** | 琴平バス株式会社が、加盟店さまと地域の皆さまのご協力をいただき運営 |
| **システム種別** | 地域CRM（顧客関係管理システム）。LINE公式アカウントベース |
| **開始時期** | 2024年2月1日 |
| **登録者数** | 2,167人（LINE友だち数、2026年3月時点） |
| **登録・利用料** | 無料 |
| **コンセプト** | 「一生に一度じゃなく、何度でも！」「観光地を訪れる」から「観光地と関わる」へ |
| **公式LINE** | https://lin.ee/UwokNEe |

### 1.2 サイトの位置づけ

琴平デジタル町民の地域CRM機能（クーポン発行、チェックイン、ポイント管理、スタンプラリー）はすべてLINE公式アカウント＋琴平ポイントアプリが担う。本Webサイトの役割は以下の3つ:

1. **サービス紹介サイト** — 「琴平デジタル町民って何？」を説明し、LINE登録（コンバージョン）に導く
2. **活動の情報発信ポータル** — イベント一覧、プロジェクト紹介、お知らせ
3. **パートナー・外部向けの窓口** — 加盟検討者への制度説明、自治体・メディアからの問い合わせ受付

### 1.3 サイトの目的

| 優先度 | 目的 | 成功指標 |
|--------|------|---------|
| 1 | デジタル町民の新規登録（公式LINE登録） | 月間登録者数 |
| 2 | 琴平町の魅力・ストーリーの伝達 | 滞在時間、直帰率 |
| 3 | イベント・お手伝い募集への参加促進 | イベント参加数 |
| 4 | パートナー事業者の新規加盟 | 問い合わせ数 |

### 1.3 ターゲットユーザー

| ペルソナ | 特徴 | 求める情報 | 導線 |
|---------|------|-----------|------|
| A. 初訪問者 | 琴平に興味を持ったばかり | 「デジタル町民って何？」 | トップ → デジタル町民とは → LINE登録 |
| B. 観光客・参拝客 | 金刀比羅宮参拝中/予定者 | 特典、クーポン、琴平への行き方 | トップ → 特典 → LINE登録 |
| C. 既存メンバー | すでにLINE登録済み | イベント情報、ポイント | トップ → 活動・お知らせ / 琴平ポイント |
| D. 担い手希望者 | 深く関わりたい人 | プロジェクト内容、お手伝い募集 | トップ → 活動・お知らせ / 琴平ポイント |
| E. 自治体・メディア・パートナー候補 | 視察・取材・加盟検討 | 実績、パートナー制度 | トップ → パートナー / お問い合わせ |

## 4. データ構造

### 4.1 イベントデータ（src/data/events.ts）

イベントは `events.ts` で管理。将来的にNotion API連携に移行予定。

```typescript
interface Event {
  id: string;
  title: string;
  date: string;
  time: string;
  location: string;
  cost: string;
  icon: string;
  summary: string;
  details: string;
  link?: string;
  linkLabel?: string;
  type: "event" | "help";  // helpはお手伝い募集（黄色ヘッダー）
  reward?: string;          // ポイント報酬（お手伝い募集用）
  capacity?: string;
}
```

### 4.2 キャラクターデータ（siteConfig.ts: townspeople）

デジタル町民キャラクター。トップページでクリックするとモーダルで紹介表示。

```typescript
{
  id: string;
  name: string;
  image: string;    // /images/character-*.png
  description: string;
}
```

現在3人: イヌカンフーさん、あっきーさん、三好さん

### 4.3 パートナー事業者（siteConfig.ts: partnerHighlights）

現在11社（テキスト名のみ）。ロゴ画像は未収集（残タスク）。
